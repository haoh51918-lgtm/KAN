# 🧭 MIRAGE-KAN 已进入 S2a：先跑通真实因子库价值链

## 🎯 为什么现在做 S2a

S1 Gate A 已确认：KAN 的半符号数值能力很强，但没有稳定产出可治理的新 HARD 原语。因此我们按 proposal 的 Plan C 前进：KAN 降为可选矿机，先回答最重要的问题——挖出的真实因子库能否在现有 Quanta 回测框架中超过 baseline。

这一步不会做图网络。图只有在因子库先产生价值、随后又发现矿机冗余问题时才有资格解锁。

## 🧱 本轮完整链路

```text
四类 typed symbolic miner
→ 256 个完整记账的候选尝试
→ 2016–2020 训练期与 2021 验证期筛选
→ 16 因子以内的不可替换因子库
→ selected / random / label-permutation 三个直接对照
→ Alpha158 当前环境复跑
→ 同一 LightGBM 500/ES50 与 TopkDropout 成本回测
→ 风险护栏与唯一阶段决定
```

`typed symbolic miner` 是“带金融类型约束的公式搜索器”；它只能生成因果、类型合法、可独立执行的抽象语法树（AST）公式。标签置换对照会在每天股票横截面内打乱标签，并重新走候选评分和选库流程，不能复用真实标签选出的库。

## 📏 S2a 预注册门槛

| 检查 | 必须达到 |
|---|---:|
| Selected 净 Information Ratio（IR，信息比率） | ≥ 0.32 |
| Selected 相对 Alpha158 replay 的 IR | ≥ +0.10 |
| Selected 相对随机合法库的 IR | ≥ +0.05 |
| Selected 相对标签置换库的 IR | ≥ +0.05 |
| 最大回撤相对 baseline 的容忍退化 | 不超过 0.03 |
| RankIC 相对 baseline 的容忍退化 | 不超过 0.003 |
| 平均换手率与实际成本倍数 | 均 ≤ 1.20 |
| 四个自然年中不差于 baseline | 至少 3 年 |

Alpha158 replay 还必须复现历史锚点：IR、最大回撤和 RankIC 的绝对偏差分别不超过 0.03、0.02 和 0.003。超差会被归类为基础设施不一致，不能拿来解释方法好坏。

## 🛡️ 证据边界

- 所有阈值已经在读取候选结果前冻结并做 SHA-256 哈希锁。
- 2022–2025 仍是 development evidence，即开发证据，不是最终盲测或 SOTA 结论。
- S2a 即使通过，也只会得到 `advance_s2_formal`，不能得到正式的 `PROMOTE_LIBRARY_VALUE`。
- S2a 若通过，才扩大到 5 个模型种子、3 个滚动候选期、19 次全流程标签置换和 10,000 次配对区块自助法。
- 因子数量、训练轮次和耗时会记录，但主横评仍是扣成本后的组合回测质量。
